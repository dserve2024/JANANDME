คู่มือการใช้งาน

ขั้นตอนง่ายๆ สำหรับสมาชิก JAN&ME

1 เริ่มต้นใช้งาน

👤 สมัครสมาชิก

  • Add Friend บัญชี LINE Official "@janandme"
  • ระบบจะสร้างบัญชีให้อัตโนมัติ
  • รอแอดมินอนุมัติ (ปกติไม่เกิน 24 ชม.)
💡 หากรอนานเกินไป กดปุ่ม "ติดต่อแอดมิน" ในหน้าข้อมูลได้เลย

🛒 เพิ่ม Shopee ID

  • ไปที่หน้า "ข้อมูล" → กดปุ่ม + ที่หัวข้อ Shopee ID
  • พิมพ์ชื่อบัญชี Shopee ที่ใช้สั่งซื้อ
  • สามารถเพิ่มได้หลายบัญชี

🏦 เพิ่มบัญชีรับเงิน

  • กดที่การ์ดบัญชีรับเงิน หรือกดปุ่ม "+ เพิ่มบัญชี"
  • เลือกธนาคาร กรอกเลขบัญชี ชื่อบัญชี
  • ใช้สำหรับรับเงินคืนจากแอดมิน
⚠️ ตรวจสอบเลขบัญชีให้ถูกต้อง เพื่อป้องกันการโอนผิด
2 การส่ง Order

📷 ส่งรูป Order จาก Shopee

  • เปิดแอป Shopee → ไปที่คำสั่งซื้อที่สำเร็จแล้ว
  • Screenshot หน้ารายละเอียดคำสั่งซื้อ
  • ส่งรูปเข้าแชท LINE Bot โดยตรง
  • ระบบจะอ่านข้อมูลจากรูปอัตโนมัติ (OCR)

🆔 ระบุ Shopee ID ให้ Order

  • หาก Order ยังไม่มี Shopee ID จะแสดงที่หัวข้อ "รอระบุ"
  • กดที่รายการ → เลือก Shopee ID จาก dropdown
  • กดบันทึก
💡 Order ที่ไม่มี Shopee ID จะไม่ถูกนำไปคำนวณยอดเงินคืน
3 ยอดเงินคืน & มัดจำ

💰 ยอดรอคืน (Refund)

คำนวณจากสูตร:

refund = subtotal + shipping
  - shippingDiscount - voucher
  - depositAmount

  • เฉพาะ Order ที่สถานะ "Completed"
  • หักยอดที่จ่ายคืนแล้วออกอัตโนมัติ

🏦 ยอดมัดจำ (Deposit)

คำนวณจาก:

deposit = จำนวนชิ้น x 100 บาท

  • เก็บมัดจำทุก Order
  • คืนมัดจำเมื่อ Order สำเร็จและตรวจสอบเรียบร้อย
4 สถานะ Order
สถานะความหมายแก้ไขได้?
Pendingส่งรูปแล้ว รอแอดมินตรวจได้
Completedตรวจสอบผ่าน รอโอนเงินคืนไม่ได้
Transferringรอค้างชำระ (กดไปหน้า Admin ได้)ไม่ได้
Transferredโอนเงินคืนเรียบร้อยไม่ได้
CanceledOrder ถูกยกเลิกไม่ได้
5 แจ้งปัญหา

🚨 วิธีแจ้งปัญหา

  • กดที่ Order ที่ต้องการ → กดปุ่ม "แจ้งปัญหา"
  • เลือกเหตุผลและอธิบายรายละเอียด
  • แอดมินจะได้รับแจ้งเตือนทันที
💡 หากมีปัญหาเร่งด่วน กดปุ่ม "ติดต่อแอดมิน" ที่หน้าข้อมูล หรือพิมพ์ข้อความในแชท LINE Bot ได้โดยตรง
